Connect to Firebase

Firebase lets all player devices sync in real time. It's free for events this size.

1Create a Google account (if needed)

Go to accounts.google.com and sign in or create a free account.

2Go to Firebase console

Visit console.firebase.google.com and click "Add project". Name it anything. Disable Google Analytics when asked. Click through until created.

3Add a web app

Click the </> icon on the project overview. Give it a nickname. Do not enable Firebase Hosting. Click "Register app" and copy the firebaseConfig block.

4Enable Realtime Database

In the left sidebar: Build → Realtime Database → Create database. Choose test mode. Click Enable.

For permanent events, use these rules in the Rules tab:

{ "rules": { ".read": true, ".write": true } }
5Paste your Firebase config below
6Host this file

Drag this HTML file onto app.netlify.com/drop for a free public URL. Share it with players.
